Jogo Tyfast

Tente acertar o mais rápido possível a palavra mostrada na tela do jogo.

Bibliotecas usadas: pygame, thread, time, english_words, random, sys, webbrowser.
Pygame foi usado para fazer o jogo num loop com suas imagens e etc, thread foi feito para fazer animações que ficam constatemente se atualizando, time foi usado para manter uma velocidade que consiga ser vista pelo monitor e olho humano, english_words foi usado para ter várias palavras do vocabulário inglês no jogo, o random foi usado para selecionar aleatóriamente alguma palavra da biblioteca english_words, sys foi usado para matar a atividade do jogo quando sair dele, webbrowser foi usado para abrir o navegador no meus links do GitHub e Replit.
